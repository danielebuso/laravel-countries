# Changelog

All notable changes to `laravel-countries` will be documented in this file.

## 1.1.0 - 2026-02-04

- Expanded dataset to include complete ISO 3166-1 list (249 entities)
- Added territories (Puerto Rico, Guam, etc.)
- Added dependencies (Greenland, Bermuda, etc.)
- Added special administrative regions (Hong Kong, Macau)
- Added other ISO-recognized entities
- Updated tests to cover new entities
- Updated documentation

## 1.0.0 - 2026-02-04

- Initial release
- 195 countries with ISO 3166-1 codes (alpha-2, alpha-3, numeric)
- Multilanguage support (English, Spanish, French, Italian, German)
- Eloquent-like query interface using Sushi
- Zero database migrations required
- Comprehensive test coverage
